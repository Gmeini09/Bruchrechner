# Bruchrechner

## Beschreibung

Dieses Java-Programm ist ein einfacher Bruchrechner.

Der Benutzer gibt zwei Brüche ein. Danach kann eine Rechenoperation ausgewählt werden:

* Addieren
* Subtrahieren
* Multiplizieren
* Dividieren

Das Ergebnis wird automatisch gekürzt ausgegeben.

Das Programm verhindert außerdem ungültige Eingaben, zum Beispiel einen Nenner mit `0`.

## Verwendete Java-Klassen

Das Programm verwendet folgende Java-Klasse:

```java
import java.util.Scanner;
```
